üìù Input Component

A flexible input component with support for various input types, validation, and styling.

Basic Input

<ui-input placeholder="Enter your name"></ui-input> <ui-input type="email" placeholder="Enter your email"></ui-input> <ui-input type="password" placeholder="Enter your password"></ui-input>

Input States

<ui-input value="Pre-filled value"></ui-input> <ui-input disabled placeholder="Disabled input"></ui-input>

API Reference

Attributes

  • placeholder - Placeholder text
  • value - Input value
  • type - Input type
  • disabled - Disable input

Events

  • input - Value changes
  • change - Focus lost with change

Methods

  • focus() - Focus the input
  • blur() - Blur the input

API Reference

Attributes

Events

Methods

üìã Select Component

A dropdown select component with customizable options and styling.

Basic Select

<ui-select placeholder="Choose an option"> <option value="option1">Option 1</option> <option value="option2">Option 2</option> <option value="option3">Option 3</option> </ui-select>

Select States

<ui-select disabled>...</ui-select> <ui-select> <option disabled>Disabled Option</option> </ui-select>

API Reference

Attributes

  • placeholder - Placeholder text
  • value - Selected value
  • disabled - Disable select

Events

  • change - Selection changes

Methods

  • focus() - Focus select
  • blur() - Blur select

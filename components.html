Input with placeholder and type:
<ui-input type="email" name="email" id="email" placeholder="Enter your email" class="form-control"></ui-input>
Password input with Bootstrap styling:
<ui-input type="password" name="password" id="password" placeholder="Password" required class="form-control form-control-lg"></ui-input>
Number input with Bootstrap classes:
<ui-input type="number" name="number" id="number" min="0" max="100" step="5" class="form-control w-25 text-center"></ui-input>
Disabled input with Bootstrap styling:
<ui-input name="disabled" id="disabled" value="This is disabled" disabled class="form-control"></ui-input>
Readonly input with Bootstrap classes:
<ui-input name="readonly" id="readonly" value="Read-only value" readonly class="form-control-plaintext border border-success"></ui-input>
Input with ALL HTML attributes:
<ui-input type="text" name="all-attrs" id="all-attrs" maxlength="10" minlength="3" pattern="[A-Za-z]+" title="Only letters allowed" autocomplete="given-name" spellcheck="false" tabindex="1" class="form-control"></ui-input>
Input with custom data attributes:
<ui-input name="custom" id="custom" data-category="user-input" data-validation="email" data-required="true" class="form-control"></ui-input>
Input with ARIA attributes:
<ui-input name="aria" id="aria" aria-label="User full name" aria-describedby="name-help" aria-required="true" class="form-control"></ui-input>
